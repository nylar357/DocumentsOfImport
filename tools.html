Malware Analysis Tools

A visual guide to popular tools for analyzing malicious software.

Wireshark

Analyzes network traffic to identify suspicious connections and data packets.

Ghidra

A free software reverse engineering suite by the NSA to decompile and analyze code.

IDA Pro

Powerful disassembler and debugger for reverse engineering malicious code.

Cuckoo Sandbox

An automated sandbox to safely run and observe malware behavior in an isolated environment.

Volatility

An open-source framework for analyzing volatile memory (RAM) dumps for forensic investigation.

PEStudio

Performs static analysis of executable files to find properties and indicators of malice.

YARA

Creates rules to identify and classify malware families based on textual or binary patterns.

Process Hacker

A real-time system monitor that can detect, suspend, and terminate malicious processes.

x64dbg

An open-source debugger for Windows, essential for analyzing 32-bit and 64-bit malware.

Burp Suite

A proxy tool for inspecting and manipulating web traffic between a browser and a server.

Process Monitor

Part of Sysinternals, it monitors file system, registry, and process activity in real-time.

Autoruns

Also from Sysinternals, it reveals programs set to run automatically during startup or login.

FLOSS

Automatically deobfuscates and extracts strings from malware binaries, even when hidden.

dnSpy

A debugger and .NET assembly editor, crucial for analyzing malware written in .NET.

Any.Run

An interactive online sandbox that allows for real-time analysis of malware in a safe environment.

VirusTotal

An online service that analyzes files and URLs using dozens of antivirus engines.
